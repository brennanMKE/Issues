## Description

Let the user drag a tab chip horizontally to reorder it. The new order is persisted (along with the existing tab list) so it survives relaunch.

## Notes

- `TabsModel.reorder(from:to:)` already exists from #0002 — it's just not wired to UI.
- `TabBarView` has a `// TODO: drag to reorder` comment marking the spot.
- Use `.onDrag { NSItemProvider(...) }` on each chip and `.onDrop` on the bar; or the `.draggable` / `.dropDestination` SwiftUI APIs (15.6+ supports them).
- Persist after every reorder by re-running the `openTabs` UserDefaults write that `TabsModel` already does on tab open/close.
