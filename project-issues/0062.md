# 0062 — German (de) localization

| | |
|---|---|
| **Status** | open |
| **Module** | App |
| **Platform** | macOS |
| **First seen** | 2026-05-05 |

## Description

Translate every key in `Localizable.xcstrings` to German. Depends on #0059.

## Translation guidance

- Use Standard German (`de`); no separate `de_AT` / `de_CH` slot for v1.
- macOS app chrome conventionally uses formal infinitive verb forms: "Ordner öffnen…", "Erneut laden", "Tab schließen".
- Match Apple's German localization conventions: "Schließen" for "Close", "Neu laden" for "Reload", "Einstellungen" for "Settings".
- German nouns are capitalized — every common noun in a translated string starts with a capital letter ("Karte" not "karte", "Zustand" not "zustand").
- Status display names: "Open" → "Offen", "In Progress" → "In Bearbeitung", "Resolved" → "Gelöst", "Closed" → "Geschlossen", "Won't Fix" → "Wird nicht behoben".
- View modes: "Swimlanes" → "Bahnen", "Timeline" → "Zeitstrahl", "List" → "Liste", "Recent" → "Letzte".
- Compound nouns can run very long (e.g. "Schlüsselbefehlsaufzeichner" for "Keyboard Shortcut Recorder"). Where this would break a fixed-width capsule or button, prefer a slightly looser translation ("Tastenkürzel" for "shortcut" is shorter than "Tastenkombination").
- Pluralization: German has `one` / `other` for most nouns, same as English. Genitive forms in interpolated strings need care — Xcode's catalog editor exposes them.

## Process

Same as #0060 / #0061: open the catalog, fill `de` slots, run with `AppleLanguages = ("de")`, walk every screen.

German strings tend to be ~30% longer than English on average — the longest of the three target locales. Confirm:

- The status-pill capsules don't truncate "In Bearbeitung".
- The view-mode capsule fits all four mode names ("Bahnen / Zeitstrahl / Liste / Letzte").
- The Settings → Shortcuts row labels fit in their column width — this is the most likely overflow site.

## Verification

- Run with system language `de` → every visible English string renders in German.
- All nouns capitalized.
- Pluralization renders correctly.
- No truncation in fixed-width chrome (status pills, view modes, settings rows).
- The shortcut recorder still says "Klicken zum Aufzeichnen" cleanly without overflowing the bordered button.

## Out of scope

- Swiss German (`de_CH`) and Austrian German (`de_AT`) — keep one `de` slot.
- Localizing date formats — handled by the system locale.
