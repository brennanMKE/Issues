## Description

Each module gets a stable accent color, derived deterministically from its name. Apply the color to the swimlane group label, the timeline row label, the list "Module" cell, and the detail panel's Module field. Helps the eye scan across views.

## Notes

- Hash the module name to an HSL hue (golden-ratio-stepped angle for good distribution); fix S/L for the dark theme so colors stay readable. ~15 lines in `Issues/Theme/`.
- A small `ModuleColor.color(for: module: String) -> Color` extension; deterministic so the same module is the same color across launches.
- Keep status colors authoritative on the bubble (status dot, badge, timeline bar fill); module color tints labels/borders only — don't fight status semantically.
- Avoid a hardcoded list of module → color mappings; modules vary per project and we shouldn't predefine them.
