## Description

Right-clicking a tab chip in the tab bar should show a context menu with: **Close**, **Close Other Tabs**, **Reveal in Finder**, **Reload**. Mirrors the conventions of Safari and Xcode tabs.

## Notes

- Add `.contextMenu` to each `TabChipView` in `Issues/Views/TabBarView.swift`.
- "Close Other Tabs" iterates `tabs` excluding `self.id` and calls `closeTab(id:)` on each.
- "Reveal in Finder" → `NSWorkspace.shared.activateFileViewerSelecting([store.folderURL])`. Works under sandbox because the URL is reached through the active security scope.
- "Reload" → `store.reload()`.
- "Close" duplicates the hover-revealed ✕ but keyboard-driven users will still want it in the menu.
