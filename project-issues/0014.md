# 0014 — Dock badge with total open issue count

| | |
|---|---|
| **Status** | open |
| **Module** | App / State |
| **Platform** | macOS |
| **First seen** | 2026-05-03 |

## Description

Show a numeric badge on the app's Dock icon equal to the total number of `open` + `in-progress` issues across all open tabs. Updates live as files change.

## Notes

- `NSApp.dockTile.badgeLabel = "\(count)"`. Set to `nil` (or empty) when count is zero.
- Recompute when any `IssueStore.issues` changes — observe across `TabsModel.tabs`. Coalesce with a tiny debounce so a multi-file edit doesn't flicker the badge.
- Decide whether the badge counts only `open` or includes `in-progress` — leaning toward including both, since both are "still on my plate".
- Add a Settings toggle later if users find it noisy. Out of scope for v1.
