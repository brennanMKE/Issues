Issues — Version 1.0

Initial release

What's new

  • Swimlane, timeline, list, and recent views for a folder of NNNN.md issue files.
  • Filters by status, module, and platform; stats bar with counts.
  • Multi-folder tabs with per-folder filters and view modes.
  • Live updates within ~150 ms when a file changes on disk.
  • Adaptive light/dark theme that follows the system appearance.
  • Localized for English, Spanish, French, and German.
  • Sparkle-based auto-updates.

Requirements

  • macOS 15.0 or later
