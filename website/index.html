Issues

A native macOS viewer for project-issues markdown — fast, focused, file-based.

Download for macOS
Version 1.0 · macOS 15.0 or later

What it does

Issues watches a folder of NNNN.md issue files and renders them as swimlanes, a timeline, a flat list, or a recent-changes view. The markdown files are the source of truth — there's no database, no sync, no cloud. Edit a file with any editor, and Issues reflects the change in about 150 ms.

It's the live dashboard for the IssuesSkill Claude Code skill: the skill writes the markdown, the app shows what's open, in progress, resolved, and closed.

Requirements

  • macOS 15.0 or later
  • Signed and notarized; no external dependencies
  • Sandboxed; only reads folders you grant it
